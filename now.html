Now

This is a now page, stolen from Derek Sivers.

Last updated: 4/7/16

Currently working as a front-end web designer at Delight. Available for freelance work. Living in Upstate New York.

I am learning how to build things using React.js and Redux. I believe the best way to learn a new technology is to just shut up and build something. So I am building a simple little application called Lemonade that tracks how often I pee. Wow, I bet you didn’t see that coming.

At Delight, we’re working on making a ticketing system to help create smoother communication between agencies and their clients. The ticketing system is being built with React and Redux. We’re also working on a project for TSL, and maintaining Ecovative’s website and ERP system.

In my free time, I’m working on redesigning shaneboland.com. I’m also working on building a front-end framework that will help me design websites faster.

I’m writing everyday, even if it’s only one sentence. I’m really starting to enjoy it. I didn’t think writing would be this fun. Blog posts coming soon once my writing is less terrible.
